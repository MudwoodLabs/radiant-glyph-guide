# Security Policy

## Reporting a Vulnerability

If you find a security issue in radiant-glyph-guide, please report it
**privately** rather than filing a public GitHub issue.

Send disclosure to: **security@mudwoodlabs.com**

Alternatively, use GitHub's private vulnerability reporting:
<https://github.com/MudwoodLabs/radiant-glyph-guide/security/advisories/new>

We aim to acknowledge receipt within **2 business days**.

## Scope

This is a **documentation repository**. Most security concerns will belong
upstream (Radiant Core, Glyph protocol, Photonic Wallet). Please report
those issues to the relevant upstream projects.

Reports about errors or omissions in the docs that could lead a reader to
make an unsafe decision (e.g., incorrect token-handling advice) are in scope.

Out of scope:

- Vulnerabilities in dependencies (report to upstream)
- Social-engineering attacks
- Issues already noted in the public issue tracker

## Disclosure

We ask for 90 days to fix before public disclosure.
